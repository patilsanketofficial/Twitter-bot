TOOL FOR IDENTIFYING AND CATEGORIZING THE TWITTER BOT'S USING MACHINE LEARNING

ABSTRACT

In today’s day-to-day life, plenty of people use Twitter in order to get new infomation as well as for entertainment purposes. Many of us are aware of the bots on Twitter. Some bots are very useful and help users to do some activities like retweeting, replaying a particular message, and so on. On the other hand, there are some bots on Twitter which harmful to users’ sensitive information. Those bots cause some crucial damage to user’s privacy. Also, some third-party users use the sensitive information of the user gathered for harmful bots for many harmful purposes. In this project, we discuss a tool that helps the user to identify which are useful bots and which are harmful bots. This tool also helps to classify bots on Twitter into two as Useful bots & Harmful bots The purpose of this tool is to provide safety measures to users while using Twitter. Also, this helps to classify the useful as well as harmful bots on Twitter in order to use Twitter safely and use the useful bots for their useful purpose and save their time as well as sensitive information.
